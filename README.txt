HCFR
----

Windows Users
-------------

Welcome to the HCFR project.

To compile the source open the solution file ColorHCFR2008.sln in a version of Visual Studio greater than 2008, allow any project conversions to happen

Select ColorHCFR as main project and build.

In order to run you will need to copy the dlls from the devlib directory into your Debug and Release directories.
