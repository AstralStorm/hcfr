HCFR
----

Welcome to the HCFR project.

To compile the source open the correct solution file for your version of Visual Studio


Visual Studio 6.0 - ColorHCFRVS6.dsw  
Visual Studio 2008 - ColorHCFR2008.sln
Visual Studio 2010 - ColorHCFR2010.sln

Select ColorHCFR as main project and build.

In order to run you will need to copy the dlls from the devlib directory into your Debug and Release directories.
