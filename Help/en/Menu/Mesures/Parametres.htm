Measures menu

 

The measures menu launches measures and handles main document parameters. Its options match buttons in Measures, Extended measures and Saturation measures toolbars.

 

Parameters

This option lets you define general measures parameters.
 


 

This option opens the following dialog box:

 

 

This dialog lets you define the number of levels in gray scale, whether gray scale is measured in IRE levels or in gray percentage, and the number of levels for near black/white gray scales, and saturations. Changing values for a filled document, or changing the IRE/% flag will reset data in corresponding grids.

Note: when using a reference document, only gris of the same size and gray scale mode (IRE/%) are comparable. The same condition is requested to perform simultaneous measures.
