HCFR Colorimeter

Select generator

 

This dialog box appears when a new file of measurements is created, or that one calls the menu Generator/Select... . It makes it possible to choose among pattern generators available.

 

 

 

View images

This choice makes it possible to use the PC to post the coloured test patterns. When the PC being used for the calibration is used as a video source on the diffuser to gauge, the generator is to be used. For the options available, see the parameters generator page “View images”.

 

HCFR Generator

This generator is integrated into HCFR Probe. This generator orders in a semi-automatic way a reader of DVD via an infra-red lamp able to emulate the actions of the remote control.

This generator must be used with the HCFR Test pattern DVD. For details abour IR codes and other options, see HCFR generator parameters.
