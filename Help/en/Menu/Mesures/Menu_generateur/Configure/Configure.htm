Display images
 

 

Blanking tab

This box determines the parameters valid for all the generators. There is only one option available:
 

   
Blank screen during measurement:

This option relates to the PC monitor used for the calibration. Indeed, to prevent that the screen of the monitor does not disturb measurements by adding parasitic light in the part, HCFR Colorimeter posts a black screen. This option is ignored if the selected generator is “Display images” and that principal display is made on the diffuser instead of PC monitor.

 

Display tab

This box determines the parameters of the generator « Display images ».
  
 

 

  1. Target screen

    DISPLAY1     
    Allows you to select the monitor on which the test pattern will be posted. The possible choices relate only to the active monitors. To have the choice of the monitor, the wide office of Windows must be activated.
     

  2. Image size (in %)  

    Expressed as a percentage to define the size of the test card of the screen. The default value of 100% is generally appropriate for all the matrix projectors, but not for the CRT (vidéo projectors and tri-tubes overhead projectors), nor with the plasma screens. It is recommended in this case to reduce the size of the test card to a lower value not to saturate the diffuser (a value of about 10 to 30% is generally appropriate).
     

  3. Display mode

    It is possible to post the test cards in GDI mode or Overlay mode. These two modes order two distinct circuits for the image processing inside the graphics board.

    GDI drawing     
    If you want to use VMR 9 , choose the GDI moode.

    Overlay      
    So, on the other hand, your favorite video player uses overlay, it is necessary to gauge your diffuser by using the test cards emitted via overlay.

  4. Overlay test

    The button “    Overlay Test” makes it possible to create a small window of overlay to the screen (a small gray rectangle clearly in top on the right of the screen). Indeed, the adjustments of overlay of the graphics board being often accessible only when the overlay is active, it can be useful to force this posting to reach the adjustments. In this case, the selected monitor must be the diffuser to be gauged; indeed, in configuration multi-monitors, the adjustments of the overlay can be different from one monitor to another. 

  5. Gray scale (GDI)

    This option defines if numeric values generated are between 0(black) and 255(white), or between 16(black) and 235(white). The first choice is a computer setting, the latter is a video setting.
