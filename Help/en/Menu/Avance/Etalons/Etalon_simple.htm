HCFR Colorimeter

 

Advanced / Calibration file

 

Build a basic calibration file

This option builds a calibration file from a measure and theoretical information about a projector colorspace.
 

 

This option launches the procedure of calibration of the sensor, which consists in posting test cards of the primary colors, white and black, in order to compare the measurements obtained with the chromacities defined in the miter “Calibration” of the configuration of the sensor.

The application can then calculate the matrix of conversion towards XYZ which defines the standard, and after a possible test of additivity indicating if it is valid, it is possible to safeguard the new standard.

The result of this calibration depends on the precision with which the projector matches the selected standard. It means it's very unreliable for a projector whose gamut is far away from the standard.

Such a calibration file shouldn't be considered reliable, and used only when no other way to obtain a true calibration file can be found.
