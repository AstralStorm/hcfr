Advanced / Calibration file

 

Build manually using reference colorimeter

This option builds a calibration file using simultaneous measures between the probe to calibrate and a reference colorimeter, which is manually driven.
 

 

This option builds a calibration file using simultaneous measures between the probe to calibrate and a reference colorimeter. It works in the same way than « Build using simultaneous measures », except you can use an unsupported colorimeter. You can also create calibration files for any tri-stimulus colorimeter when the Activate calibration files on all tri-stimulus probes is checked in advanced preferences.

This operation consists in measuring primary colors and white using HCFR colorimeter (or another tri-stimulus probe) and at the same time, you must take measurements with another colorimeter, and note results from this reference colorimeter. Changing current test pattern is a manual operation, using the Return key. After measuring primary colors and white, ColorHCFR request values measures by the reference colorimeter in the colors from reference colorimeter dialog box. You can input values in XYZ or xyY coordinates. After validating data, the calibration file is computed, a signature is requested, the additivity information and white delta E is displayed, and you can save the new calibration file.
