Toolbars...
 

This box allows you to define special behaviour of mouse button on the view toolbar.

 


 

While clicking on “Toolbars”, the “Preference” dialog box appears like this.
 

 
 

 

This box lets you define actions to perform when right clicking of wheel clicking on a button of the view toolbar. You can define one of the following action for each mouse button:

  • Open the view in a new window : This option opens a new global windows for the document, like with Window/New Window menu option, which will contain the view corresponding to the clicked button.

  • Activate view in all opened windows : This option activates the view in all existing document window. This is practical when analyzing differences between many documents.

  • Open a new view even if it already exists (duplicate) : This option a view, even if it already exists. It is equivalent to the Duplicate tab option in tab contextual menu. It can be useful if you want twice the same view with different options.

  • Close the view : Close all tabs corresponding to the button in current window.
