Référence...
 

This box allows you to define the values of reference used during calculations colorimetric.
 

While clicking on “Reference”, the “Preference” dialog box appears like this.
 

 
 

Several options are available:
 

  • Colors

 

      1. Standard 

        The combo box "Standard" allows you to define the standard used. That determines the standard gamut (position of the primary colors on diagram CIE).

        Three posible options :

        PAL/SECAM         
        Format PAL/SECAM (Phase Alternating Line/Séquentiel Color with Memory), used in Europe for hertzian television.

        NTSC         
        NTSC         (National Television System Committee) is a standard of coding of the video color developed at the in the USA in the year 1950

        HDTV         
        it's the standard used for HD

        sRGB         
        it's the standard used for digital photo.

         

      2. White

        Illuminant D65
        The combo box "White" allows you to define the white of reference. It is starting from this white that are calculated the variations in RGB levels graph and the Delta E, but also the values posted in control Levels RGB and Target.

         

 

  • Gamma reference

 

This option defines how the average gamma is computed

 

      1. Display gamma

      2. Display gamma with black compensation

      3. Camera gamma

      4. Camera gamma, manual offset

      5. Optimized

 
 

 

    • Reference Gamma

  

The Reference Gamma defines the reference curve in luminance graph.
