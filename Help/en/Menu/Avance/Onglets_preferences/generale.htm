HCFR Colorimeter

General...
 

This box allows you to define the general parameters operation of the application HCFR Colorimeter.

 


 

While clicking on “General”, the “Preference” dialog box appears like this.
 

 

 

 

  • Application

    The checkbox     "Multiple instances" allows you to authorize the launching of several authorities application. When it is not notched, the application cannot be launched once. Operations of opening of the files starting from the explorer Windows thus open a new document inside the existing application instead of a new opening each time.

    Note     : the change of this parameter is taken into account only after restarting HCFR Colorimeter.

     

  • User measurement

    In the group users Measurements, the checkbox "    Dispaly test color...."  allows to activate the use of the generator to post the color of test chosen at the time of free measurements.


    When the checkbox “    Continuous measuring” is notched, the free measurements launching button look changes.


    Instead of a     camera when allowing a single measurement, it becomes a green arrow allowing to launch uninterrupted measurements. During the process of measurement, the button becomes a red square allowing to adopt the measures. When measurements are launched uninterrupted, the window of combined graph appears automatically.


    Detect primary colors during user measures :

    When this option is checked, free measures are analyzed to see if they are near a reference primary or secondary color. In this case, the matching primary or secondary color measure in the document is replaced by the free measure. It makes possible to use continuous free measures to setup primary and secondary colors, and then see them change the CIE diagram in real time.     

    When the continuous mode is selected, that modifies the behavior of the test pattern generator. Indeed, in continuous mode, the colors of tests are posted only if the selected generator is “View Images”, and if it is configured to display the test cards on another screen which is the principal window HCFR Colorimeter. In this case, the principal screen is not masked any more when the generator appears (contrary to unit free measurements, when the blanking is activated). The test card is posted on the secondary screen during the time measurements, and the posted color can be modified in real time by using the window of colors test.

     

  • Dynamic iris


    When the checkbox “    Beep at the beginning .....” is notched, a beep sound is emitted just before the measurement starts. This beep allows to materialize the end of the latency times, which allows to control visually what is measured corresponds well to the good test card, and which the iris is finished adjusting.

     

  • Colors to add when measuring primary and secondary colors:

    This option allows adding a white measure, and eventually a black measure, to each primary and secondary colors series. The white is often used, for calibration files, XYZ adjustment, delta luma, and delta Y/Lux on primaries and secondaries. It is of very few interest to refuse measuring additionnal white. Black is unemployed, but is could be useful one day. It is useless to activate this option, except for archive if a future version uses it one day.

     

  • Reference colors for saturation graphs

    Use measured colors (when available) : When checked, this option uses measured primary and secondary colors instead of colorspace reference colors to draw Saturation-shift curves. If there are no measures for primary and secondary colors, color references defined in preferences/references are always used, regardless this option.
