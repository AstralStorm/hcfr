Test colors

This option opens the Test Color window. This control is useful to define a color which must be displayed by the generator when performing free measures (unique or continuous).
 


 

You can change the color using the Color Picler, or by clicking on any point in the CIE diagram. The selected color RGB components are displayed upon the colored background.
