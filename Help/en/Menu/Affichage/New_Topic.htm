HCFR Colorimeter

 

View Menu

 

Luminance

This option opens a tool window displaying in real time the luminance measured by LX-1108 luxmeter, if it is correctly linked to the PC and configured. The value displayed is the same than indicated in status bar. This window is useful only when you need to see this value in big, for example when you are far from your PC while manipulating the probe or the diffuser.
