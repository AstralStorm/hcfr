Software

  • The HCFR Colorimeter  functions only under Windows Vista, Windows XP and Windows 2000.

  • A USB cable (not provided with the sensor HCFR Colorimeter ).

  • It is possible to install several cables USB end to end (according to the quality of cables, length tested is 8 meters).
