HCFR Colorimeter

 

Components
 

Primary colors

You can see the luminance curves of each red, green, blue component by checking the options in contextual menu.
 

 

Four options are available.
 

  1. Luminance

    The luminance graph displays luminance values computed from near white gray scale measures. This curve is analogous to gray scale, and shows thin luminance details near white level.
     

  2. Red 

    Display red component curve.
     

  3. Green

    Display green component curve.
     

  4. Blue

    Display blue component curve.
