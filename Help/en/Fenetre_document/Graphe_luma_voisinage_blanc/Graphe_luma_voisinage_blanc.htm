Near white luminance graph

The near white luminance graph shows in details the luminances near 100% gray, by step of 1% or 1IRE

This curve is analogous to gray scale luminance, but show only near white levels. Those measures are represented only on this graph, which is useful only to optimize general contrast setting (white level). The number of measured points can be changed, and is 5 by default. The most interesting values are 98, 99 and 100. During calibration, the goal is to have the highest contrast setup which shows a luminosity difference between the last levels. When the contrast setting is too high, you have a characteristic curve of burned white, ie a setting which does not allow the projector to show the subtle luminance difference in lightest parts of image, which results in a curve like that:



 

Warning: it's easy to have a beautiful curve with a low contrast setting, which lowers the projector white level, overall luminosity and overall contrast.

This graph can be displayed in logarithmic mode to enhance readability, and eventually to refine the high-gamma setting, for projectors which have it. 

The options relative to this graph are available using right mouse button. They can define components, reference curves, define scale, graph parameters, and save graph in a file.
