Généralités

 

The document window contains many tabs, each tab matching a different view. You can open simultaneously many windows for the same document, using the Window/New window menu. This way, you can see many different tabs at the same time.

When creating a new document, or opening a measure file, the document window opens with only one opened tab, the Measures one.

To open a new tab, there are many possibilities :

- The Graphs menu.

Each option opens and activates the chosen tab in current document window. If the tab already exists, it's simply activated..

- The tab tray popup menu. A right click on tab tray (outside existing tabs) or a click on button opens the Graphs menu.

- The Views toolbar.

Each button matches a Graphs menu option. The buttons are hilighted when corresponding tabs are opened in current document window. Some options in Preferences / Toolbars let you assign special functions to those buttons using a right click or a wheel click..

Each tab can be activated simply by clicking on it, to make the underlying view visible..

A right click on a tab opens the tab contextual menu :

  • - Activate tab : Symply activates tab, exactly like an ordinary click

  • - Activate tab on all documents : Activate the tab on all existing document windows, and create it on all documents which do not contain it already. This function is practival to compare graphs on different documents. This function is also activated by clicking on the tab with the Control key down..

  • - Duplicate tab : Create a new tab identical to the clicked one. This can be useful if you need twice the same graph with different internal options (scale, mode...)

  • - Open tab in a new window : Create a new document window, as you could do using Window / New window menu, with the view matching the tab inside..

  • - Close tab : Delete the tab and close the underlying view. This function is grayed when there is only one tab in the document window. By the way, the document window cannot exist without any tab. To close the last tab, you must close the document window. A tab can also be closed using the button at the extreme left on the tab tray..

The tab tray contains two checkboxes at its right, Adjust XYZ and Reference.

Those checkboxes replicate the same checkboxes in the Measures view, to make them accessible immediately even when another view is visible. Their behaviour is documented in the Measures view documentation.

You can put the tab tray on top of document window, using View / Tab control on top menu option.

The document window size depends on which views are inside. When the Measures view is present, the window cannot be smaller than this view requested size. But when only graphical views are present, you can reduce document window to a very small size..

You can change current tab using additional mouse buttons, used for back/next functions in many applications, and also using keyboard Alt key with left and right arrows.

 

The available tabs are the following:

 

Measures view

Luminance

Gamma

RGB levels

Color temperature

CIE diagram

Near black gray scale

Near white gray scale

Saturation - luminance

Saturation - shifts

Combined graph - free measures
