Components
 

Primary colors

You can see the luminance curves of each red, green, blue component by checking the options in contextual menu. It can be useful to see a difference between two components, but usually, you should prefer using the RGB levels graph, which is more readable and easy to interpret.

 

 

Four options are available.
 

  1. Luminance

    The luminance graph displays luminance values computed from gray scale measures. In X axis, the gray levels are shown in percentage or in IRE levels, with one point per measure. On Y axis, the luminance is expressed in percentage of maximum luminance, obtained at 100% gray or 100 IRE. Point 100 is always displayed at 100% luminance.
     

  2. Red 

    Display red component curve.
     

  3. Green

    Display green component curve.
     

  4. Blue

    Display blue component curve.
