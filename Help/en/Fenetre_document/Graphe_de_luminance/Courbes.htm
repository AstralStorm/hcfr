Reference Curves

The contextual menu allow reference curve display, reference document curves, and average curve display.   
 


 

The three first options concern the additional curves which can be displayed.

 

  1. Display reference 

    When checked, display a reference curve, which depends on reference gamma selected in Préférences/Références.
     

  2. Display reference measure  

    This option displays curves matching reference document measures. Those curves are displayed in dotted lines, with the same color than current document curves. It allows easy comparison between two series of measures in two different documents.
     

  3. Display average 

    When checked, this option displays a curve matching the average gamma values. This average is computed from measures, and have an ideal reference curve shape. This curve shows immediately which parts of the measured curve is under or below an acceptable gamma.
