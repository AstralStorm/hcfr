HCFR Colorimeter

 

Generalities

 

This view contains all measures data under numeric format. It's divided in three main parts. The top part contains all information concerning measures.

 

View Group

Select the measures series displayed in the gris using the combobox in View group. The Editable data checkbox allow manual entry inside grid cells. This checkbox is grayed when display unit is « xyz », and when the Adjust XYZ checkbox is checked.

The up and down arrow right Editable data let you increase or decrease the number of visible lines in grid, which is fine when using delta lines related to reference document.

 

Sensor Group

The sensor group indicates the selected sensor, and the button open its parameters dialog. This button matches  Measures/Sensor/Configure… menu

 

Generator Group

The Generator group indicates the selected pattern generator, and the buttons opens its parameters dialog. This button matches the Measures/Generator/Configure… menu.

 

Parameters Group

The parameters group contains two checkboxes, which affects globally the measures. Those checkboxes are duplicated at right of document window tab tray, to be immediately accessible when another tab is active.

  • The « Reference measure » checkbox selects the reference document, which is unique across the whole application. When checked, all groups appear oranged, to distinguish easily the reference document. When there is a reference document, all curves from this document are added on other documents curves, with the same colors, but as dotted lines (under option in their contextual menu : « Display reference measure » ). It makes it easy to compare two documents with different measures. It concerns all graphs, except the combined graph for free measures. Warning: if the number of measured points is not the same in the reference document and another document, or if the gray scale unit choice in gray percents or IRE level is different, the reference curves will not be displayed on the second document. The horizontal scales must be exactly identical to allow comparisons. The reference document graphs do not display references curves (which would superpose normal curves).

  • The « Adjust XYZ » checkbox activates the XYZ correction matrix. It is active only when an adjustment matrix has been defined. For details about coordinates adjustment, see Advanced / Adjust XYZ menu.

 

Measures grid group

The group containing measures grid have a changing name regarding displayed data, selected from the combobox in View group. When Gray Scale is selected, the gamma average and on/off contrast are displayed. When gamma calculation is "Optimized (regression)", the computed offset value associated with gamma is also displayed.

The grid contains a variable number of lines. The three fist are colorimetric coordinates, whose unit varies depending on selected radiobutton in the Display group. Those three first lines are the only one editable when the Editable Data checkbox is checked, the others are read only. The following lines indicate the Delta E and delta xy from the reference. The delta E calculus can take the target luminance into account or not. For details about this calculus, see advanced preferences.

If there is a reference document, two supplementary lines are added, to indicate Delta E and Delta xy between measure and reference document matching measure.

For Gray Scale, a Gamma Y line is added. This line contains the theoretical value of luminance Y computed for each gray level, excepted zero and 100 levels. This value is computed from white luminance measure, in function of the reference gamma defined in preferences, using preferred gamma calculation mode (to use black compensation or offset value). Using this value, you can for example change the projector gamma settings using continuous measures on a gray test pattern, until you attain the theoretical value.

For primary and secondary measures, a delta luma line is appended. This line display the delta between attended luminance for the primary or secondary color related to measured white, regarding the colorimetric reference defined in Preference/references. This information is not displayed if white has not been measured along with primary and/or secondary colors (which is the case when this measure has been canceled in Preferences/general, or when opening a file generated by a version 1.x). The values are displayed in percentage, plus or minus, an absence of delta being displayed with the equal « = » sign.

Note : between different color spaces, the delta luma on primary colors, mainly the blue, can be very important. It means if high deltas are displayed, it can indicate you measure a projector setup upon a HD colorspace with a SD colorspace reference, or the opposite. Changing the colorspace in preferences/references refresh immediately the delta luma, which can ensure this hypothesis.

If a luxmeter is linked and activated in advanced preferences, another lined is appended, indicating the values measured by the luxmeter during colorimetric measurements. For Gray Scale and primary/secondary colors, a second line indicates the percentage of luminance shift between luxmeter and colorimeter, considering white as 100% reference. For details about luxmeter use, see Advanced preferences.

 

Particularities :

Some arrays are somewhat particular, and need some more remarks.

  • Primary and secondary colors : This array contains 6 primary and secondary colors, plus white and black. Those two last colors are measured optionally, regarding the Colors to add when measuring primaries and secondaries in preferences/general. White is used to compute delta luma, but also in all matrix computation (calibration files, XYZ adjustment, controlled mode measures). If white is not present, those computations will be done without using it, and the results will be less precise. Black measure is totally unemployed in computations, but it may be useful in a future version. It is useless to measure it in normal conditions. Note: black and white in primary/secondary colors array are the same measures than black and white in contrast array.

  • Contrast : This array contains four measures: black and white (for on/off contrast), ANSI 1  et ANSI 2, which are the black and white cells measured during ANSI contrast. Contrast values are displayed in right grid part, if measures are available.

  • The number of measures for Gray Scale, Near Black and Near White gray scales, and Saturations, can be set in the Measures parameters dialog, which is opened using the Measures / Parameters menu. This dialog let you choose if gray scale is in gray percentage or in IRE levels.

  • Free measure array has a variable size, depending on the number of measures performed. There is a maximum of 250 measures, after that, first measures are deleted to make room. The delta E and delta xy in free measures are computed relatively to reference white, or relatively to one of the three reference primary colors, or relatively to one of the three secondary colors computed from white and primaries. By default, white is the reference, but when the measured color is near a primary or secondary color, this one is taken as reference. This is highlited by a colored background, red, green, blue, yellow, cyan or magenta, in the delta E and delta xy cells instead of usual neutral gray.

 

Display Group

The Display group let you select colorimetric unit in which values are displayed in the grid. The "Sensor" unit displays raw probe data. This choice is not accessible when the probe has no calibration file, because it returns directly XYZ values.

Buttons Group

The Go button launches the measures displayed in grid. It substitute to the matching option in Measures menu regarding selection in the View group combobox. It's icon changes to match the measures it calls. When performing free measures, it can be continuous measures (green arrow) or unique measure (camera), depending on the option Continuous measuring in preferences/general.

A Control+Click on this button runs simultaneous measures on all documents, regarding data displayed in the grid. It matches one of the options in the Advanced/Simultaneous measures menu. In the case of free measures, only a unique free measure can be launched in simultaneous mode (there is no continuous measures in simultaneous mode).

The Delete button cancels data in the array. It's the only way to clear measures, there is no matching menu option. Concerning free measures, this button allow deleting only selected columns, or the whole data if there is no selection.

 

The down-left part contains information about selected color.

The RGB levels indicate percentages of red, green and blue in measured color. The numerical values are the same than in RGB levels graph.

The target indicated where is the selected color relatively to the reference color. Generally, it is the reference white, but when the selected color is near a primary or secondary color, this one is taken as reference. The target background then becomes red, green, blue, yellow, cyan or magenta to indicate the reference is not white.

Note: the target disappears when the document window is too small to display it entirely.

This data grid gives information about selected color in usual colorimetric units.

 

The down-right part of this view contains annexes.

You can select the type of information displayed using the combobox upon the variable zone. By default, it's a free text zone used to comment the measures (this text is saved in measures files).

There are two other particular choices:

- Target : This is a target control duplication which is described above, in a bigger size. This can be useful when doing a first setting on two gray test patterns, if you need to be far away from PC screen to change projector settings.

 

- Spectrum : Display selected color spectrum. The spectrum is available only when using a spectrophotometer (Eye One Pro).

 

Other choices correspond to often used graphs. Those graphs are identical to the ones displayed in other tabs, but it can be fine to have a reduced version directly in measures view.

The little checkerboard button, which appears only when selected data in grid is Contrast, displays the ANSI test pattern (it is useless if test pattern is not displayed by PC). The goal is to display the checkerboard on the projector in order to center the probe at the center of a square zone (white or black), before running ANSI contrast measuring.
