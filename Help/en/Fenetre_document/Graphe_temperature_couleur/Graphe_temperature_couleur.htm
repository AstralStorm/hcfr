Color temperatur graph

This graph is an accessory allowing to appreciate the quality of a set-up. It relies on the grey scale's measurements, and displays, for each IRE level and its color temperature. The color temperature should be the same for all the IRE levels, and as close as possible to the reference which is 6500 Kelvin.

A color temperature higher than 6500 K shows an excess of blue compared with red. At the opposite, a color temperature lower than 6500 shows an excess of red, still compared with blue. Green component has a limited influence on the color temperature.

In fact, a given temperature is related to a curve, not a dot, in the color space. Consequently it is possible to measure a 6500 K "grey" which is actually green, or at the contrary, a magenta.
 

   

Because of this, the color temperature histogram is more a control tool a posteriori than a tool meant for adjustment. For the calibration, the RGB levels histogram is far more suitable cause it displays more informations.

However, for the novices, consulting this histogram may help understand RGB levels because it shows any inbalance between red and blue in a very straight forward way and easier to apprehend than balancing the three components simultaneously.

Once red and blue are balanced, adjustment of the green level are then easier to conceive in checking the RGB level histogram, wherein blue and red are already balanced. 

 

The options relative to this graph are available using right mouse button. They can define reference curves, define scale, graph parameters, and save graph in a file.
