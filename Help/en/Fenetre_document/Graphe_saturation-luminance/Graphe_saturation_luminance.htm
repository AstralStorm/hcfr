Saturation luminance graph

Saturation - luminance graph allows checking of the brightness constancy for a color whose saturation only changes. Primary and secondary colors are shown in this graph, with as many dots as saturation levels were measured. This graph is purely informative as they are not any adjustment available on diffuser allowing correction of any default. Therefore, you should just check all curves are straight horizontal lines. Actually, inappropriate adjustments may generate back side effects, subject to disturb these curves.

 

 

 

The options relative to this graph are available using right mouse button. They can define scale, graph parameters, and save graph in a file.
