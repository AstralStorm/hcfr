Generalities

 

Combined Graph - free measures

The combined graph contains four distinct parts: luminance, RGB levels, delta E and color temperature. The luminance, delta E and color temperature graphs can be removed using contextual menu.

The last free measures are taken into account for this graph. Each measure is displayed in all graphs, the points of a unique measure being vertically aligned. During continuous measuring, the graph scrolls from right to left while data are furnished by the probe.

 

The number of horizontal points can be configured at 10, 20 or 50 using contextual menu.

The luminance graph display luminance values in arbitrary units (Y). Unlike gray scale luminance graph, the unit cannot be in percentage, because we do not have a reference white. This graph scale automatically adjust in real time to avoid maximum value goes out of scale. For the same reason (missing reference), it is not possible to perform black compensation, nor display a logarithmic mode.

RGB levels graph, delta E and colors temperature are exactly identical to gray scale graphs.

The options relative to this graph are available using right mouse button. They can define scale, graph parameters, and save graph in a file.
