Near black luminance graph

The near black luminance graph shows in details the luminances near 0% gray, by step of 1% or 1IRE

This curve is analogous to gray scale luminance, but show only near black levels. Those measures are represented only on this graph, which is useful only to optimize general brightness setting (black level). The number of measured points can be changed, and is 5 by default. The most interesting values are 0, 1 and 2. During calibration, the goal is to have the lowest black level setup which shows a luminosity difference between the first levels. When the black level setting is too low, you have a typical black blocked curve, ie a setting which does not allow your projector to display subtle details in dark zones. The resulting curve looks like that :

 

Warning: it's easy to have a beautiful curve with a high brightness setting, which "grays" the projector blacks, and reduce overall contrast.

This graph can be displayed in logarithmic mode to enhance readability, and eventually to refine the high-gamma setting, for projectors which have it. This mode is available only if a reference white has been measured, either with gray scale, either with primary/secondary colors, or during a contrast measure.

The options relative to this graph are available using right mouse button. They can define components, reference curves, define scale, graph parameters, and save graph in a file.
