Generalities

 

RGB levels graph

This graph is the main color calibration tool. It relies on the grey scale's measurements, and displays, for each gray level, relative level of each component in percentage. When every values are at 100%, it means the grey's color is the same as the D65's point on the CIE diagram, thus the color calibration is perfect.
 

   

The levels displayed are relative to each others, that means if a components is too high, at least one of the others levels would be too low. Therefore, during the set-up of the diffuser, you should seek minimizing most important differences in relation to the 100% reference, on the incriminated components. As an example, if you have 150% blue, and 95% red and green, adjusting the excess of blue would increase relative level of red and green to 100%. Percentages displayed are also independent upon brightness adjustment. Brightness, contrast and gamma adjustments won't change the percentage (except sometimes on the margin, dots at the graph's boundary : IRE 0 and 100).

Values used for displaying the histogram are also shown in the RGB level measurements window, which shows relative levels in relation to the selected color in the measurements window.

In general, you should find gain and offset adjustments for each components on every diffuser.

Gain works as a contrast adjustment, but for only one component. Offset works as a brightness adjustment for only one component. It sets brightness level around IRE zero.

In theory, the offset is used to set the « height » of the corresponding component's curve and the gain is used to set the « gradient » of the curve, in order to a have a straight horizontal line at the right level. Despite this, between the theory and the practice, experience shows these adjustments have often side effects one each other. Actually, many adjustments and measurements are needed to fully understand how a change affects all the curves and to obtain the expected result.

 

The options relative to this graph are available using right mouse button. They can define reference curves, define scale, graph parameters, and save graph in a file.
