Reference Curves

The contextual menu allow reference curve display, reference document curves, and Delta E activation.   

 

 

The three first options concern the additional curves which can be displayed.

 

  1. Display reference 

    When checked, display a reference curve, which is simply an horizontal line matching 100% level.

  2. Display reference measure  

    This option displays curves matching reference document measures. Those curves are displayed in dotted lines, with the same color than current document curves. It allows easy comparison between two series of measures in two different documents.

  3. Delta E curve 

    When checked, the delta E graph is appended under RGB levels graph. The Delta E measures the difference between measured color and target color (D65 point on CIE diagram) in subjective units. The more the measure is far from D65, the more the delta E grows. A delta E lower than or equal to 3 means color difference is invisible to human eyes. Calibration is then perfect. A delta E lower than or equal to 6 is considered very satisfying. One of the main calibration goals is to have a Delta E curve with all points under 6

    The Delta E curve is not a setting tool, its only a control tool. It does not indicate in which direction to go, nor which setting should be modified on the projector. It can juste measure the distance between measured color and a target, and the quality of a particular calibration.

    The Delta E computation can use target luminance or not depending on an option. For details on this option, see advanced preferences.
