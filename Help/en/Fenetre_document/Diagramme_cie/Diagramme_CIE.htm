CIE Diagram

The CIE diagram displays gray scale points, primary colors, secondary colors, saturation levels and free measures on the normalized CIE diagram, with some useful references. 

The diagram can be displayed in CIE xy or CIE u'v' mode, regarding contextual menu

Below the diagram in  CIE u'v' mode.
 

 

 

The options relative to this diagram are available using right mouse button. They can define displayed points, diagram background, and save diagram in a file.
