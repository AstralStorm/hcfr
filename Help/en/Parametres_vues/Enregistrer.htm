Save Option

This option allows you to save graphs in image files.
 


 

The "Graphs saving" dialog box opens when you click "Save".
 


 

  1. Image size 

    Window size      
    Save image using actual window size.

    300*200 / 600*400     
    Dimensions in pixels of the image to save.

    User Width
    Type the size you want for the image.
     

  2. File type 

    JPEG     
    "    Joint Photographic Experts Group" This format is more compressed than GIF format. It is less precise but generally it's enough.

    BMP
    BMP, for "    Bitmap", is an uncompressed format.

    PNG
    "    Portable Network Graphics" It's a compressed format comparable to GIF, with better compression ratio.

     

Note: The graph image in the file will have a black or white background regarding option "Save chart files with a white background" in preferences / appearance.
