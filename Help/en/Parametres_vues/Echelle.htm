HCFR Colorimeter

 

Scale
 

Generalities

This menu let you modify graph scale. This menu slightly changes from one graph to another.
 


 

Auto adjust

Scale is optimized to make curves fill entire window size.
 

Direct values or sub-menus

Many options are available to manually define a scale.
 


 

  1. 1-3 

    Values on Y axis (    vertical) are between default values.
     

  2. Auto-adjust 

    Graph scale is optimized.
     

  3. Zoom + 

    Increases graph scale.     

  4. Zoom - 

    Decreases graph scale.

  5. Shift Up 

    After using "    Zoom +" or "Zoom -", you can shift curves up without changing scale.
     

  6. Shift Down 

    After using "    Zoom +" or "Zoom -", you can shift curves down without changing scale.
     

 

Values

You can manually define values for both scale.
 


 

  1. Vertical scale 

    Y axis values (    vertical) . 



     

  2. Horizontal scalee 

    X axis values (    horizontal).
