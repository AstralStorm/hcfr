Graphs parameters

This option let you change appearance of graphs.
 

 

It opens the "Graphs Parameters" dialog.
 


 

  1. Graphs

    Graph name
    Select the curve you want to change. The list of available curves depends on the graph from which this dialog is called.


    Parameters
    Available options :
    - Curve color
    - Display dots
    - Display tooltips
    - Line thickness.
    - When thickness is equal to 1, you can select line style

     

  2. Global



    Change graph environment
    Display or not labels, axis, etc...
