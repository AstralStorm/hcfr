Colorimètre HCFR

Installation du logiciel Colorimètre HCFR
 

Procédure d'installation

En double cliquant sur le fichier télécharger précédemment, la boite de dialogue suivante apparaît.

  • Cliquez sur enregistrer. 
     


 

  • En cliquant sur Oui la fenêtre suivante s'ouvre :
     

 
 

Le bouton au sujet de vous permets d'avoir des information complémentaires.
 

  • En cliquant sur le bouton suivant vous arrivez sur la fenêtre demandant votre acceptation concernant le contrat de licence.
    Veuillez le lire attentivement avant de l'accepter.
    Ce contrat de licence peut être imprimé via le bouton "Imprimer la licence"
     



 

  • En cliquant sur le bouton suivant vous arrivez sur la fenêtre "Avant propos" donnant quelques informations complémentaires.

     


 

  • En cliquant sur le bouton suivant vous arrivez sur la fenêtre "Installation" permettant le choix du répertoire d'installation du logiciel.
    Le répertoire par défaut se trouve sous "     Program files/Colorimètre HCFR" vous pouvez le modifier en cliquant sur parcourir.
     


 

  • En cliquant sur le bouton suivant vous arrivez sur la fenêtre vous permettant éventuellement les fichiers étalons .thc et mesures .chc à installer.
    Vous pourrez toujours télécharger ultérieurement le fichier que vous aurez omis.

     



 

  • En cliquant sur le bouton suivant vous arrivez sur la fenêtre vous permettant éventuellement de créer un raccourci dans le menu Démarrer.
     

 

  • En cliquant sur le bouton suivant vous arrivez sur la fenêtre vous permettant éventuellement de créer un icône sur le bureau de votre ordinateur.
     


 

 

L'icone se trouvant sur vote bureau.


 

  • En cliquant sur le bouton suivant vous arrivez sur la fenêtre listant les différents fichiers qui vont être installés.

     

 

  • En cliquant sur le bouton suivant vous arrivez sur la fenêtre de fin d'installation. vous pouvez exécuter le logiciel Colorimètre HCFR sans redémarrer votre ordinateur.

 

 

  • En cliquant sur le bouton suivant vous arrivez sur la fenêtre de fin d'installation. Vous pouvez maintenant exécuter le logiciel Colorimètre HCFR sans redémarrer votre ordinateur.
