 ++++++  Informations complémentaires

 

Le pilote USB doit déjà se trouver sur votre ordinateur. Si vous avez auparavant téléchargé et installé le logiciel Colorimètre HCFR, le pilote USB se trouve dans le répertoire du même nom sous le répertoire d'installation du logiciel.



Sinon vous pouvez le télécharger en suivant ce lien: Pilote USB .

 

Procédure d'installation de la sonde.
 

  • Raccorder le  câble USB à la sonde puis à l'ordinateur. La led verte sur le dessus de la sonde clignote.
    Après quelque secondes, la fenêtre suivante s'ouvre.
     


 

  • Cocher la case d'option Installer à partir d'une liste.
    Cliquez sur suivant.
     



 

  • La fenêtre suivante s'ouvre.
     


 

  • Cocher la case d'option inclure cet emplacement dans la recherche.
    Cliquez sur parcourir.
    La  fenêtre     recherche un dossier s'ouvre, selectionnez le répertoire pilote USB sous le répertoire d'installation du logiciel ColorimètreHCFR.
    Lorsque le répertoire est sélectionné, cliquez sur le bouton     OK de la fenêtre recherche un dossier.
     


 

 

  • La  fenêtre se met à jour avec le chemin du pilote USB nécessaire à l'installation de la sonde.Cliquez sur parcourir.
    Cliquez sur suivant.
     


 

 

  • Wiindows recherche le pilote USB afin de procéder à son installation.
     


 

 

  • Wiindows vous informe de la fin de l'installation du pilote USB. La led verte cesse de clignoter et reste allumer en permanence.
