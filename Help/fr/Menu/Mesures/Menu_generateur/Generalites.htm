Configuration

Ce menu permets d'avoir accés aux paramétres de configuration et sélection des trois générateurs actuellement disponibles:

Affichage mires: génération des mires par cette application pour effectuer l'étalonnage.
   
Générateur HCFR: génération des mires par certains équipements compatibles avec cette fonction.
 
DVD Manuel: génération des mires en utilisant le DVD spécifiquement prévu pour effectuer l'étalonnage avec cette application.
 

L'apparence de la boite de dialogue " Configure" dépend de l'option choisie dans le sous menu "Sélectionne" dans "Générateur".
