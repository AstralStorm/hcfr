Colorimètre HCFR

Affichage des mires
 

 

Onglet "Blanking"

Cette boîte détermine les paramètres valables pour tous les générateurs. Il n’y a qu’une seule option disponible :
 

   
Eteint l’écran pendant la mesure :

Cette option porte sur le moniteur du PC utilisé pour l’étalonnage. En effet, pour éviter que l’écran de monitoring ne perturbe les mesures en ajoutant de la lumière parasite dans la pièce, Colorimètre HCFR affiche un écran noir. Cette option est ignorée si le générateur sélectionné est « Affichage mires » et que l’affichage principal se fait sur le diffuseur à calibrer.

 

Onglet "Display"

Cette boîte détermine les paramètres du générateur « Affichage mires ».
 


 

 

  1. Ecran cible

    DISPLAY1     
    Permet de sélectionner le moniteur sur lequel les mires seront affichées. Les choix possibles ne concernent que les moniteurs actifs. Pour avoir le choix du moniteur, le bureau étendu de Windows doit être activé.
     

  2. Taille de la mire en (%) 

    Permet de définir la taille de la mire en pourcentage de l’écran. La valeur par défaut de 100% convient généralement à tous les projecteurs matriciels, mais pas aux CRT (vidéoprojecteurs et rétroprojecteurs tri-tubes), ni aux écrans plasmas. Il est recommandé dans ce cas de réduire la taille de la mire à une valeur plus faible pour ne pas saturer le diffuseur (une valeur de l’ordre de 10 à 30% convient généralement).
     

  3. Display mode

    Il est possible d’afficher les mires en mode GDI ou en mode Overlay. Ces deux modes commandent deux circuits distincts pour le traitement de l’image à l’intérieur de la carte graphique.

    GDI drawing     
    Si vous utilisez VMR 9 (ou autre version), préférez le mode GDI.

    Overlay      

    Si par contre votre player vidéo favori utilise l’Overlay, il faut calibrer votre diffuseur en utilisant les mires émises via l’Overlay.

     

  4. Test Overlay

    Le bouton « Test Overlay » permet de créer une petite fenêtre d’overlay à l’écran (un petit rectangle gris clair en haut à droite de l’écran). En effet, les réglages d’overlay de la carte graphique n’étant souvent accessibles que lorsque l’overlay est actif, il peut être utile de forcer cet affichage pour accéder aux réglages. Dans ce cas, le moniteur sélectionné doit être le diffuseur à calibrer ; en effet, en configuration multi-moniteurs, les réglages de l’overlay peuvent être différents d’un moniteur à l’autre.
     

  5. Echelle de niveaux de gris (GDI)

    Cette option permet de définir si le générateur sort des valeurs numériques comprises entre 0(noir) et 255(blanc) ou entre 16 et 235. Le premier choix correspond à un mode de sortie informatique, le second à un mode vidéo.
