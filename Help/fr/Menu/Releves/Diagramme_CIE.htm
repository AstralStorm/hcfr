Diagramme CIE

Le graphe de diagramme CIE permet d’afficher les points de l’échelle de gris  avec diverses références utiles.

Le diagramme CIE est une représentation graphique en forme d'aile qui représente le lieu de toutes les longueurs d'ondes visibles.

Les couleurs pures ou totalement saturées se trouvent sur le contour du spectre et sont 100% pures.

Le rouge est situé dans le coin inférieur droit, le vert à la pointe supérieure et le bleu dans le coin inférieur gauche du diagramme.

La ligne droite qui relie les deux extrémités du spectre est appelée "la droite des pourpres".On peut situer le lieu du corps noir dans la zone centrale, appelée "source". La courbe représente la couleur d'un corps noir théorique lorsque la température s'élève de 1000K à l'infini.

Le point A est la couleur d'une ampoule à filament de tungstène remplie de gaz (T = 2856K).
La source B représente le soleil au méridien. Le point C représente un ciel couvert à midi (blanc de référence de la télévision).
Le point C représente un ciel couvert à midi (blanc de référence de la télévision).

La source D (le fameux D65) correspondant à un corps noir rayonnant à T = 6504K est un blanc un peu plus vert qui sert de référence en matière d'étalonnage vidéo. 

 
Ci dessous le diagramme est affiché avec le fond.

Le fait de cliquer sur le graphe ci dessous vous renvoie directement vers l'aide concernant les vues graphiques du diagramme CIE.
 


 
 

Les différentes options inhérentes à ce diagramme sont disponibles via le bouton droit de la souris.
