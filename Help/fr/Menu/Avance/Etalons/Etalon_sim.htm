Fabriquer un étalon à partir de mesures simultanées

Cette option permet de fabriquer un étalon en effectuant des mesures simultanées avec le colorimètre à étalonner et une sonde de référence.
 

 

Cette option permet de fabriquer un étalon en effectuant des mesures simultanées entre le capteur HCFR et un colorimètre de référence. Elle permet également de créer des étalons pour tous les colorimètres tri-stimulus quand l’option Active les étalons pour toutes les sondes tri-stimulus est cochée dans les Préférences, onglet Avancé.

Pour l’utiliser, vous devez obligatoirement disposer d’un colorimètre de référence supporté par le logiciel, et d’un colorimètre HCFR. Il faut avoir deux documents ouverts simultanément, le document lié au colorimètre de référence ayant la checkbox « Mesure de référence » cochée. Il faut ensuite appeler cette fonction avec le document lié à la sonde HCFR (ou une autre sonde tri-stimulus) actif (la fonction renvoie un message d’erreur si le document d’appel est le document de référence).

Les deux capteurs étant positionnés face à l’écran, à une distance équivalente, sans se masquer mutuellement, il suffit d’appeler cette option pour lancer la mesure des couleurs primaires et du blanc. Une fois les mesures terminées, le logiciel calcule l’étalon, puis affiche une boîte de Signature vous demandant un nom ou un pseudonyme pour signer l’étalon (ce n’est utile que si vous diffusez vos étalons ou que vous les transmettez à HCFR, afin de pouvoir en identifier l’origine).

Ensuite, le programme affiche une boîte de message indiquant l’additivité et le delta E entre le blanc mesuré par le colorimètre de référence, et le blanc calculé avec le nouvel étalon.

Les données d’additivité sont purement informatives, des écarts importants pouvant être rencontrés sur certains modèles de projecteurs sans que cela n’invalide l’étalon. Par contre, le Delta E du blanc calculé à partir de l’étalon et le blanc mesuré par le colorimètre de référence est important. Une valeur trop importante invalide l’étalon. On peut considérer qu’un Delta E inférieur à 3 est acceptable pour l’étalon. En principe, si le blanc a été mesuré, le Delta E pour le blanc doit être égal à zéro. Si ce n’est pas le cas, c’est que l’étalon n’a pu être calculé qu’en employant les couleurs primaires.

Après cet affichage, le logiciel propose de sauver l’étalon dans un fichier. Si vous ne sauvez pas le fichier à ce stade, vous pourrez le faire ultérieurement à l’aide de la fonction Sauver le fichier étalon.

Il convient ensuite de vérifier la pertinence du nouvel étalon en effectuant des mesures simultanées de l’échelle de gris entre les deux documents, et d’utiliser les divers diagrammes pour vérifier qu’il y a peu d’écarts entre les résultats donnés par le colorimètre de référence et la sonde HCFR.

Lorsque vous disposez d’un nouvel étalon pour un projecteur pour lequel aucun étalon n’est fourni, merci de le transmettre à l’équipe de développement via le forum du colorimètre, sur le site www.homecinema-fr.com. Afin de nous permettre de le diffuser, merci de communiquer avec cet étalon les deux fichiers .chc de mesures simultanées ayant permis la génération de l’étalon.

Note : pour « jouer » avec le logiciel, il est possible d’étalonner le Capteur Simulé, et également de l’utiliser comme colorimètre de référence. Les étalons générés par ce biais n’ont bien entendu aucune valeur, et vont varier d’une fois sur l’autre puisqu’ils sont basés sur des valeurs aléatoires.
 

 

La boîte de signature:

 

La boîte permettant de signer un étalon apparaît ainsi:

 

 

Cette boîte apparaît automatiquement lors de la génération d’un étalon par mesures simultanées ou manuellement à partir d’un colorimètre de référence. Elle permet de saisir un nom ou un pseudonyme qui sera stocké en tant que nom d’auteur dans le fichier étalon. Cette information sera visible dans l’onglet Matrice Capteur de la boîte de configuration du capteur.

 

La boîte d'additivité

 

Les informations d'additivité apparaissent sous forme d'une message box.

 

 

Ces informations sont reprises dans les commentaires associées à l'étalon.
