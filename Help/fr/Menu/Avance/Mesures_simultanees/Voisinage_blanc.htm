Colorimètre HCFR

 

Menu Avancé / Mesures simultanées

 

Préambule

Ce menu permet d’effectuer des mesures simultanées entre plusieurs colorimètres supportés par le logiciel. Pour cela, il faut disposer d’autant de documents ouverts qu’on a de colorimètres différents reliés au PC.

Les colorimètres du commerce ne peuvent être supportés qu’à hauteur d’un seul par modèle. Il peut y avoir plusieurs sondes HCFR pilotées simultanément, mais dans ce cas, il faut renseigner manuellement le port COM de chaque sonde pour que le logiciel puisse les différencier (les ports COM réels peuvent être trouvés dans la liste des périphériques connectés à l’ordinateur).

Pour le capteur simulé, il est possible d’en mettre plusieurs (à des fins de test uniquement, ceux-ci n’ayant bien entendu aucun intérêt pratique).

Le générateur utilisé est celui du document actif quand le menu est appelé.

 

Voisinage du blanc

Cette option permet de mesurer simultanément le voisinage du blanc sur tous les documents ouverts.
 


 

Le résultat des mesures du voisinage du blanc est affiché dans le  tableau des mesures et dans le graphe de voisinage du blanc.

 

 

 

Note : la led des colorimètres HCFR impliqués dans ce processus reste éteinte pendant toute la durée des mesures, à condition qu’elles aient un firmware de version 5.20 ou ultérieure.
