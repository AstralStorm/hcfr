Colorimètre HCFR

 

Toolbars...
 

Cette boîte permet de personnaliser le style d’affichage de l’application Colorimètre HCFR. Il est possible de modifier le style des contrôles et des menus ainsi que les couleurs employées.
 


 

En cliquant sur l'onglet "Toolbars" , la boite de dialogue se présente ainsi.
 

 
 

Les options de cette boîte ne modifient en rien les résultats des calculs de l’application.

 

Cet onglet permet de définir les actions à effectuer lors d’un click droit ou un click du bouton central sur les boutons de la toolbar vues. Pour chaque bouton, il est possible de choisir l’une de ces quatre actions :

  • Ouvre la vue dans une nouvelle fenêtre : Cette option permet d’ouvrir une nouvelle fenêtre globale pour le document, comme avec le menu Fenêtre/Nouvelle fenêtre, qui contiendra uniquement le graphe correspondant au bouton cliqué.

  • Active la vue dans toutes les fenêtres ouverts : Cette option permet d’activer la vue correspondant au bouton cliqué dans toutes les fenêtres document ouvertes. Cette fonction est pratique lors de l’analyse des différences entre plusieurs documents de mesures préalablement juxtaposés.

  • Ouvre une nouvelle vue même s’il en existe déjà une : Cette option permet de créer plusieurs vues identiques dans la fenêtre. C’est équivalent à l’option Duplique l’onglet du menu contextuel des onglets. Cela peut être utile si on veut pouvoir disposer en même temps de deux vues avec des options internes différentes.

  • Ferme la vue : Ferme tous les onglets correspondant au bouton cliqué dans la fenêtre courante.
