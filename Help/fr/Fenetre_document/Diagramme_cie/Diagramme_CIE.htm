Diagramme CIE

Le graphe de diagramme CIE permet d’afficher les points de l’échelle de gris, les couleurs primaires, les couleurs secondaires, les niveaux de saturation, et les mesures libres sur le diagramme normalisé CIE, avec diverses références utiles. 

Le diagramme peut être affiché en mode CIE xy ou CIE u'v' en fonction du menu contextuel

 

Ci dessous le diagramme est affiché en mode CIE xy.
 

 
 

Ci dessous le diagramme est affiché en mode CIE u'v'.
 

 

 

Les différentes options inhérentes à ce diagramme sont disponibles via le bouton droit de la souris. Elles permettent de définir les points affichés, le fond du diagramme, et d'enregistrer le diagramme dans un fichier.
