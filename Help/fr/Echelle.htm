Colorimètre HCFR

Installation du ColorHCFR

Généralités.

  • Le ColorHCFR fonctionne sous Windows XP, 2000 et Vista.

  • Le cable USB n'est pas fourni.

  • Il est possible d'installer plusieurs câbles bout à bout (suivant la qualité des câbles,  longueur testé 8 mètres) .

  • Télécharger le pilote.

 

Installation.

  • Brancher le colorimètre sur une prise USB.

  • Suivre les instructions de Windows qui demande d'installer le pilote.
